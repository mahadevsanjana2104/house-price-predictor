# 🏠 House Price Prediction App

This is a full-stack machine learning web app that predicts house prices based on features like bedrooms, bathrooms, living area, and floors.

## 🚀 Features
- ML model using Linear Regression
- Flask backend API
- Interactive frontend UI
- Real-time prediction

## 🛠 Tech Stack
- Python
- Flask
- Scikit-learn
- HTML, CSS, JavaScript

## ▶️ How to Run
```bash
pip install -r requirements.txt
python app.py
